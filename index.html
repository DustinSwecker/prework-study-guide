HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head can contain elements such as: <meta>; <title>; <style>; <base>; <link>; <script>; etc.
  • The body element can contain elements such as: <a>; <h1>; <section>; <div>; <br>; <img>; etc.
  • Attributes modify elements or describe elements. Some examples of attributes are src, href, id, and width. For more on attributes visit W3 school's attribute reference site.
  • Sementic HTML elements such as <section> have names which describe the function and content of the particular element and are generally preferable to elements such as <div> and <span> which provide no additional information about function and content.
  • The <h1> tag should appear only once per page.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style an html document: inline CSS--code inserted directly into an element; internal css stylesheet--code inserted in the <head> element; and external CSS stylesheets--the creation of a CSS document which is linked to the HTML in the <head> element.
  • Most HTML styling is done via external CSS stylesheets in accordance with the Seperation of Concerns principle which states that when possible code should be seperated into seperate files based on their function.
  • HTML elements are essentially boxes and their design is modified by altering the margin, padding, and borders in CSS.
  • The display value in CSS allows us to alter the behavior of HTML box elements. For example, the block display value makes it so: a box breaks onto a new line; the width and height values are respected; and the padding, margin, and border values will cause other elements to be pushed away.
  • * is used on a CSS stylesheet to signify CSS values which will effect all elements on a page.
  • Class attributes allow us to designate CSS styles to specific selectors.
  • Once a CSS stylesheet is complete, link it to your HTML by creating a link element within the head element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git config: accesses and alters git configuration
  • git add: adds modifications to the current working branch
  • git commit: adds commit to git
  • git diff --check: checks for white space errors
  • git pull origin main: pulls the main branch to the local
  • git push origin branch-name: pushes the local branch file to the github server
  • For more on GitHub commands visit this cheat sheet.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a dynamic language as variables are not tied to a particular data type.
  • A string is a data type consisting of a sequence of text contained by quote marks.<
  • A Boolean data type is a either true or false and requires no quote marks.
  • An array is a data type which allows multiple values to be held in a single variable.
  • Operators are math symbols which produce a result based on two values.
  • Conditionals are code structure used to test if an expression is true or not.
  • A very common conditional is an if...else statement.
  • Falsy is a term referencing the values of 0, -0, 0n, "", undefined, null or False.Truthy represents all other values.
  • A for loops uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed repeatedly.
  • For loops contain 3 statements: a starting point; a condition; and a statement defining the number of iterations.
  • Functions are repeatable code sequences which are only used if called in the code structure.
